**TOLONG JANGAN LANGSUNG PUSH KE MASTER**
Cara pasang project:
1. Install & configure git
2. Buat folder baru
3. ketik ```git init```
4. ketik ```git remote add origin [url repo]```
5. ketik ```git pull origin master```
6. copy file ```config.php``` dan ```database.php``` ke direktori ```/application/config/```
